# Research SOP
1. Acknowledge the research request.
2. Use `Bash` (e.g., `curl` or python scraping scripts) or a connected Web Search MCP to gather data.
3. Synthesize the data in `SCRATCHPAD.md`.
4. Present the findings using the "BLUF" (Bottom Line Up Front) method.
5. Cite sources (URLs) for all factual claims.
