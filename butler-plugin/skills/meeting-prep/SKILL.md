# Meeting Preparation SOP
1. Identify the meeting from `~~calendar`.
2. Extract attendees.
3. Query `~~knowledge base` (Notion) for the attendees' names to find past context or relationships.
4. Query `~~email` for the last 2 threads with these attendees.
5. Output a concise 4-part Briefing:
   - **Who:** [Names & Titles]
   - **Context:** [Summary of last email/interaction]
   - **Goal:** [Inferred purpose of meeting]
   - **Action Required:** [e.g., "Read the attached PDF before joining"]
